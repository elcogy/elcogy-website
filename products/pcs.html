PCS 储能变流器

双向 AC/DC 变换器，实现电池与电网之间的能量转换，支持并网/离网双模式

核心功能

  • ✓ 高效率能量转换，最大效率≥98%
  • ✓ 宽电压范围，适配多种电池系统
  • ✓ 低谐波失真，THDi≤3%
  • ✓ 支持并联扩容，最大支持 10 台并联
  • ✓ 并网/离网无缝切换，切换时间≤10ms
  • ✓ 支持 V/f 控制、PQ 控制、下垂控制

技术参数

  • 额定功率：50kW / 100kW / 250kW / 500kW
  • 输入电压：DC 500V ~ 900V
  • 输出电压：AC 380V / 400V / 480V（三相）
  • 电网频率：50Hz / 60Hz
  • 功率因数：-1 ~ +1 可调
  • 冷却方式：智能风冷
  • 防护等级：IP54

应用场景

  • • 工商业储能系统
  • • 源网荷储一体化项目
  • • 微电网系统
  • • 光储充电站
  • • 用户侧储能电站
