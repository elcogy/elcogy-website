工商业储能柜

集成化储能系统，即插即用，适用于工厂、商场、数据中心等场景

核心优势

  • ✓ 模块化设计，支持灵活扩容
  • ✓ 智能温控系统，确保电池最佳工作温度
  • ✓ 多重安全防护（电气/消防/环境）
  • ✓ 10 年 + 使用寿命，全生命周期成本最优
  • ✓ 即插即用，现场安装≤2 小时
  • ✓ 静音设计，运行噪音≤55dB

产品规格

  • 标准容量：100kWh / 200kWh / 500kWh / 1MWh
  • 系统电压：DC 400V / 800V
  • 充放电倍率：0.5C / 1C 可选
  • 循环寿命：≥6000 次（@80% DOD）
  • 防护等级：IP54（室内）/ IP65（室外）
  • 工作温度：-20℃ ~ +50℃

应用场景

  • • 工厂削峰填谷，降低用电成本
  • • 商场需量管理，减少基本电费
  • • 数据中心备用电源，保障关键负荷
  • • 充电站配储，提升容量利用率
  • • 光储一体化，提升自发自用率
